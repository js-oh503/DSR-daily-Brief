# 조선업 일일 동향 프로그램 설정 방법

초보자도 따라할 수 있도록 단계별로 설명합니다.

---

## 1단계: Python 설치 확인

명령 프롬프트(cmd)를 열고 아래를 입력하세요:
```
python --version
```
버전이 표시되면 OK. 설치가 안 됐다면 https://www.python.org/downloads/ 에서 설치하세요.

---

## 2단계: 필요한 라이브러리 설치

명령 프롬프트에서 이 폴더로 이동 후 실행:
```
cd C:\Users\Admin\Desktop\workspace
pip install -r requirements.txt
```

---

## 3단계: Claude API 키 발급

1. https://console.anthropic.com 접속 → 회원가입/로그인
2. 왼쪽 메뉴 **"API Keys"** → **"Create Key"** 클릭
3. 키를 복사해 둡니다 (sk-ant-... 로 시작하는 문자열)

---

## 4단계: API 키 환경 변수 설정 (Windows)

### 방법 A: 영구 설정 (권장)
1. **시작 메뉴** → "환경 변수" 검색 → **"시스템 환경 변수 편집"**
2. 하단의 **"환경 변수(N)"** 클릭
3. "사용자 변수" 영역에서 **"새로 만들기(N)"** 클릭
4. 변수 이름: `ANTHROPIC_API_KEY`
5. 변수 값: 복사해 둔 API 키 붙여넣기
6. 확인 → 확인 → 확인
7. **명령 프롬프트를 새로 열어야 적용됩니다**

### 방법 B: 그때그때 설정 (테스트용)
명령 프롬프트에서 실행할 때마다 아래 입력:
```
set ANTHROPIC_API_KEY=sk-ant-여기에API키붙여넣기
python shipbuilding_daily.py
```

---

## 5단계: 프로그램 실행 테스트

```
python shipbuilding_daily.py
```

정상 실행되면 `reports` 폴더에 HTML 파일이 생성되고 브라우저가 자동으로 열립니다.

---

## 6단계: 매일 아침 자동 실행 설정 (Windows 작업 스케줄러)

1. **시작 메뉴** → "작업 스케줄러" 검색 → 실행
2. 오른쪽 패널 **"기본 작업 만들기..."** 클릭
3. 이름: `조선업 일일 뉴스` → 다음
4. 트리거: **"매일"** → 다음
5. 시작 시간: `07:00:00` (원하는 시각) → 다음
6. 동작: **"프로그램 시작"** → 다음
7. 프로그램/스크립트: `python`
8. 인수 추가: `C:\Users\Admin\Desktop\workspace\shipbuilding_daily.py`
9. 마침

> ⚠️ 주의: 자동 실행 시 API 키가 환경 변수에 등록되어 있어야 합니다 (방법 A 사용).

## 자주 묻는 질문

**Q: "ModuleNotFoundError" 오류가 나요**
→ `pip install -r requirements.txt` 를 다시 실행해보세요.

**Q: "ANTHROPIC_API_KEY 환경 변수가 설정되지 않았습니다" 오류가 나요**
→ 4단계를 다시 확인하고, 명령 프롬프트를 새로 열었는지 확인하세요.

**Q: 기사가 0건 수집되어요**
→ 인터넷 연결을 확인하거나, 잠시 후 다시 실행해보세요. 일부 RSS 피드는 일시적으로 접근이 안 될 수 있습니다.

**Q: 비용이 얼마나 드나요?**
→ Claude Haiku 모델을 사용하며, 하루 한 번 실행 기준 약 $0.01~0.03 (약 15~45원) 수준입니다.
